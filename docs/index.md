## 📦 安装

```bash
npm install @weblif/fast-ui --save
# or
yarn add @weblif/fast-ui
```

## 🔨 示例

```js
import { Button } from '@weblif/fast-ui'

const App = () => (
  <>
    <Button type="primary">PRESS ME</Button>
  </>
)
```
